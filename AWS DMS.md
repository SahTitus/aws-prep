# AWS Database Migration Service (AWS DMS)

**AWS DMS** migrates 
- **relational databases**
- **data warehouses**
- **NoSQL databases** 
- **and other data stores from on premises to the AWS Cloud. You can use AWS DMS**

## Which problem does AWS Database Migration Service (AWS DMS) solve?
- **Minimizing application downtime during database migrations**
- **A key benefit of AWS DMS is that the source database remains fully operational during migration, which minimizes downtime to applications.**
