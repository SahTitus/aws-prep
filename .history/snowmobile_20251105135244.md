### AWS Snow Family (Snowball and Related Devices)

# AWS Snowball*
A data transport and edge computing device offered by Amazon Web Services used to transfer large amounts of data between on-premises environments and AWS without relying on the internet.
It helps migrate terabytes to petabytes of data securely, efficiently, and cost-effectively.

# Key Insight:
*When internet transfer is too slow or expensive, AWS ships you a physical device ‚Äî you load your data, return it to AWS, and they upload it to the cloud.*

| Device                              | Primary Use               | Capacity      | Edge Compute      | Typical Scenario             |
| :---------------------------------- | :------------------------ | :------------ | :---------------- | :--------------------------- |
| **Snowcone**                        | Small-scale edge device   | ~8 TB         | Yes               | IoT, remote sites            |
| **Snowball Edge Storage Optimized** | Large-scale data transfer | ~80 TB usable | Optional          | Data migration, backups      |
| **Snowball Edge Compute Optimized** | Edge compute workloads    | ~42 TB usable | Yes (EC2, Lambda) | Edge AI, analytics           |
| **Snowmobile**                      | Massive data transfer     | Up to 100 PB  | No                | Entire data center migration |
