# AWS Migration Tools and Frameworks Summary
| **Resource / Tool**                          | **Description / Purpose**                                                                                                                            |
| -------------------------------------------- | ---------------------------------------------------------------------------------------------------------------------------------------------------- |
| **Three Phases of Migration**                | The three main phases of a large-scale AWS migration are **Assess**, **Mobilize**, and **Migrate & Modernize**.                                      |
| **AWS Cloud Adoption Framework (AWS CAF)**   | Provides best practices and tools to **accelerate cloud adoption** and align business, people, and processes during migration.                       |
| **Seven Migration Strategies (7 Rs)**        | The seven common strategies for cloud migration: **Relocate**, **Rehost**, **Replatform**, **Refactor**, **Repurchase**, **Retain**, and **Retire**. |
| **Migration Evaluator**                      | A **migration assessment tool** that helps build a **business case** for AWS using data-driven analysis and cost projections.                        |
| **AWS Application Discovery Service**        | **Discovers on-premises servers**, configurations, and dependencies to create detailed migration plans.  . It helps customers simplify, **expedite**, and **reduce the cost of migrating** and **modernizing applications.**                                            |
| **AWS Application Migration Service (MGN)**  | Simplifies and automates **lift-and-shift migrations** of applications from on-premises or other clouds to AWS.                                      |
| **AWS Migration Hub**                        | A **central dashboard** to track, coordinate, and manage migration progress across AWS and partner tools.   To **assessment**, **planning**, and **implementation of your migration**. It provides **tools**, **guidance**, and **automated recommendations** to collaborate with your team and **track your migration**.                                          |
| **AWS Database Migration Service (AWS DMS)** | Enables **fast and secure migration of databases** to AWS with minimal downtime; supports heterogeneous migrations.                                  |
| **AWS Schema Conversion Tool (AWS SCT)**     | Converts **database schemas and code objects** between different engines (e.g., Oracle ‚Üí PostgreSQL).                                                |
| **AWS DataSync**                             | Automates and accelerates **secure online data transfers** between on-premises storage and AWS.                                                      |
| **AWS Transfer Family**                      | Provides **managed file transfer** over **SFTP, FTPS, and FTP** into AWS storage services like **S3** or **EFS**.                                    |
| **AWS Direct Connect**                       | Establishes a **private, dedicated network connection** from your data center to AWS for low-latency, high-bandwidth data transfer.                  |
| **AWS Snow Family**                          | Offers **offline data migration** using physical devices (like **Snowball Edge**) for large-scale or disconnected environments.                      |
