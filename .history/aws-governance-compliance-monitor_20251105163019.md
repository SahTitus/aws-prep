| **Service / Resource**      | **Description**                                                                                                               |
| :-------------------------- | :---------------------------------------------------------------------------------------------------------------------------- |
| **Amazon CloudWatch**       | Monitors AWS resources and applications in real time for performance, utilization, and operational health.                    |
| **AWS CloudTrail**          | Records user activity and API calls for auditing, security monitoring, and troubleshooting (“Who did what, where, and when”). |
| **AWS Artifact**            | Self-service portal for on-demand access to AWS compliance reports, certifications, and agreements.                           |
| **AWS Config**              | Assesses, audits, and evaluates AWS resource configurations for compliance.                                                   |
| **AWS Audit Manager**       | Continuously audits AWS usage to simplify risk and compliance assessments.                                                    |
| **AWS Organizations**       | Centrally manages and governs multiple AWS accounts and policies at scale.                                                    |
| **AWS Control Tower**       | Enforces and manages governance, compliance, and security rules across all AWS accounts.                                      |
| **AWS Service Catalog**     | Creates and manages a curated catalog of approved AWS resources for consistent governance.                                    |
| **AWS License Manager**     | Manages software licenses and optimizes licensing costs across AWS and on-prem.                                               |
| **AWS Trusted Advisor**     | Provides recommendations to improve cost efficiency, performance, security, and resilience.                                   |
| **AWS Health**              | Notifies about events, outages, and changes affecting your AWS resources.                                                     |
| **AWS IAM Access Analyzer** | Helps set and verify permissions to achieve least privilege security.                                                         |
