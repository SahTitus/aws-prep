# AWS Snow Family — Quick Reference

| Device | Capacity (HDD) | Capacity (SSD) | Primary Use Case |
|:--|:--|:--|:--|
| **AWS Snowcone** | Up to 8 TB | Up to 14 TB | Small-scale data transfer, edge computing, remote locations |
| **AWS Snowball Edge** | Up to 80 TB | Not applicable (storage optimized version is 80 TB HDD) | Larger data transfer, edge computing, and storage in locations with limited network |
| **AWS Snowmobile** | Up to 100 PB | Not applicable | Massive data center migrations, exabyte-scale transfers |



| Device           | Storage | Compute | Main Use              |
| :--------------- | :------ | :------ | :-------------------- |

| Snowball Storage | 80 TB   | Low     | Bulk transfer         |
