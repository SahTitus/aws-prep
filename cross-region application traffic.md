# Which AWS service or feature allows the user to manage cross-region application traffic?
- A. Amazon AppStream 2.0
- B. Amazon VPC
- C. Elastic Load Balancer
- D. Amazon Route 53

**Correct answer: D. Amazon Route 53**

**Amazon Route 53** provides **global DNS-based traffic routing**, which allows you to distribute application traffic across **multiple AWS Regions** using features like:

* Latency-based routing
* Geolocation routing
* Weighted routing
* Failover routing

The other options do **not** manage cross-region application traffic.
